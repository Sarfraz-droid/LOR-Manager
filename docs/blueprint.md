# **App Name**: LoR Tracker Pro

## Core Features:

- Professor Profile Creation: Create and manage professor profiles, including contact information and areas of expertise.
- LoR Request Logging: Log individual Letter of Recommendation requests, including application deadlines and professor assignments.
- Status Tracking: Track the status of each LoR request (e.g., requested, in progress, submitted).
- Application Linking: Link LoRs to specific university/scholarship applications.
- Deadline Reminders: Set deadlines for each LoR request and receive automated reminders.
- Intelligent Recommendation Suggestion: Suggest which professor a student should ask for a Letter of Recommendation based on factors such as areas of study, research focus and courses taught.
- Data Persistence: Persist data across sessions.

## Style Guidelines:

- Primary color: Deep blue (#2E4A62) to evoke an academic and professional feel.
- Background color: Cream (#F5F5DC) for a soft, professional backdrop.
- Accent color: Gold (#D4AF37) to highlight key information and CTAs, adding a touch of elegance.
- Body and headline font: 'Literata', a serif, to give a literary and formal feel to the body and headers.
- Use minimalist, professional icons to represent actions and statuses.
- Design a clean, organized layout with clear hierarchy to manage professor profiles and LoR requests efficiently.
- Subtle transitions and animations to indicate state changes and improve user experience.
